# 🎯 军事理论刷题 · 网页版

一个专为 **军事理论课程** 设计的移动端刷题工具，支持 **练习模式** 与 **考试模式**，帮助你高效复习、查漏补缺。  
直接运行在浏览器中，所有数据保存在本地，保护隐私。

> **因为题库格式复杂，所以存在部分提取错误和一些未识别题目，仅供参考！

---

## ✨ 核心功能

- **练习模式**  
  可按题型（单选/多选/判断）、完成状态（全部/未做过/错题重练）筛选题目，支持自定义题数、起始题号、随机顺序。  
  答题后即时反馈对错，并自动记录错题与已做题进度。

- **考试模式**  
  自动从题库中抽取：**20道单选（1分）+ 20道多选（2分）+ 10道判断（0.5分）**，满分 50 分。  
  支持题目预览、任意跳转、随时交卷，交卷后显示详细得分与逐题正误，可快速定位错题复盘。

- **进度记忆**  
  练习进度自动保存（浏览器本地存储），下次打开可无缝继续。  
  错题本和已做题记录持久化，方便针对性复习。

- **智能选项解析**  
  兼容题库中选项合并的情况（如一条字符串包含 `A. xxx B. xxx`），自动拆分为独立选项，确保按钮点击正常。

- **美观轻量**  
  响应式设计，适配手机与电脑，界面清爽、交互流畅，按钮带有触感反馈。

## 📖 使用方法

1. **下载 / 复制代码**  
   将完整的 `index.html` 文件保存到本地（或直接复制代码）。

2. **开始使用**  
   双击 `index.html` 用浏览器打开即可，所有功能直接可用。
